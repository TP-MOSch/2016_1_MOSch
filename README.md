Kill The Birds!
==========


Браузерная казуальная action-игра с элементами стратегии и RPG.
Выполняется в рамках проекта по предметам "Фронтенд" и "Углубленное программирование на Java" студентами **МГТУ им. Н. Э. Баумана**.
